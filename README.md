# University Dormitory & Restaurant Management System

A C++ OOP project built with Qt for managing university dormitories,
rooms, students, and restaurant services.

## Features
- Student management (add/remove/assign to rooms)
- Dormitory and room management with capacity tracking
- Restaurant menu management per dormitory
- Statistics dashboard showing occupied/available rooms
- Data persistence via text files
- Graphical interface built with Qt

## OOP Concepts Used
- Composition (Dormitory owns Restaurant and Rooms)
- Inheritance (Student inherits from Person)
- Polymorphism (virtual display() function)
- Encapsulation (private members, public getters/setters)

## How to Run
1. Clone this repository
2. Open `UniversityDormitoryandRestaurantManagement.pro` in Qt Creator
3. Build the project (Ctrl+B)
4. Run (Ctrl+R)
